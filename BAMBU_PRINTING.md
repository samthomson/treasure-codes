# Bambu Studio Multi-Color Printing Guide

## Quick Setup

1. **Import Files**
   - Open Bambu Studio
   - Import `treasure_XX_base.stl` (green)
   - Import `treasure_XX_qr_pattern.stl` (white)

2. **Position Objects**
   - Both files should align automatically
   - QR pattern sits on top of base
   - Ensure both are at Z=0

3. **Assign Colors (AMS Required)**
   - Select base object → Right-click → Change filament → Green
   - Select QR pattern → Right-click → Change filament → White
   - Or use modifier meshes (see README)

4. **Print Settings**
   - Layer height: 0.2-0.3 mm
   - Infill: 20-30%
   - Supports: Off
   - Enable multi-color mode

## Tips

- The QR pattern is designed to sit flush on the base
- White squares are 1mm tall for good contrast
- Text "https://treasures.to" is embossed on the green base
- Test print one first to verify alignment
